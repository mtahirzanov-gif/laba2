# Лабораторная работа №2: Система расчета зарплат

## Описание
Программа для учета работников, видов выполненных работ и автоматического расчета заработной платы в IT-компании.

## Функциональность
- ✅ Добавление сотрудников с указанием должности
- ✅ Учет выполненных работ по типам (программирование, тестирование, дизайн и др.)
- ✅ Настройка почасовых ставок для каждого вида работ
- ✅ Расчет зарплаты сотрудника на основе отработанных часов
- ✅ Поиск сотрудников по фамилии
- ✅ Вывод детальной информации и общей суммы выплат
- ✅ Валидация пользовательского ввода

## Технические детали
- **Язык программирования:** C++
- **Стандарт:** C++11 и выше
- **Среда разработки:** Microsoft Visual Studio
- **Структура программы:**
  - `enum WorkType` — виды работ (программирование, тестирование, дизайн, документация, менеджмент)
  - `enum Position` — должности (разработчик, менеджер, тестировщик, дизайнер, аналитик, стажёр)
  - `class Employee` — модель сотрудника с историей работ
  - `class RateManager` — управление почасовыми ставками
  - `class PayrollManager` — главный менеджер (паттерн Singleton)

## Как запустить в Visual Studio

### Вариант 1: Открыть существующий проект
1. Запустите **Visual Studio**
2. Нажмите **Файл** → **Открыть** → **Проект или решение**
3. Выберите файл `.sln` или `.vcxproj` (если есть)
4. Нажмите **F5** или кнопку ▶️ **Локальный отладчик**

### Вариант 2: Создать новый проект и добавить файл
1. **Файл** → **Создать** → **Проект** → **Пустой проект (C++)**
2. Укажите имя проекта (например, `laba2`) → **Создать**
3. В **Обозревателе решений**: ПКМ по папке **Исходные файлы** → **Добавить** → **Существующий элемент**
4. Выберите `main.cpp` из этого репозитория
5. Нажмите **F5** для запуска

### Вариант 3: Быстрый запуск
1. Создайте пустой проект C++
2. Добавьте `main.cpp`
3. Проверьте стандарт C++: **Свойства проекта** → **C/C++** → **Язык** → **Стандарт языка** → `ISO C++11` или новее
4. Нажмите **Ctrl+F5** для запуска без отладки
